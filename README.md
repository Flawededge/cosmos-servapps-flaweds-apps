# Cosmos ServApps - Flawed's Apps

Personal Cosmos market source for custom ServApps.

After GitHub Pages is enabled for this repository, add this source in Cosmos Market:

```text
https://flawededge.github.io/cosmos-servapps-flaweds-apps
```

The generated app list is published at `https://flawededge.github.io/cosmos-servapps-flaweds-apps/servapps.json`.

Build the local catalog files with:

```sh
node index.js
```
